# Sharad's Portfolio

🚀 A dark/light-themed, responsive personal portfolio built with React.js & Tailwind CSS, following the Dracula theme.

## 📂 Features

- ✅ **Dark & Light Theme Support** – Toggle between dark (Dracula) and light mode
- ✅ **Fully Responsive** – Works on all devices
- ✅ **Smooth Navigation** – Scrolls to sections with active highlights
- ✅ **Projects Showcase** – Interactive project cards with GitHub & Live links
- ✅ **Contact Section** – Direct links to GitHub, LinkedIn, and Email

## 🛠 Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Icons**: React Icons

## 🚀 How to run on localmachine ?

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/SharadJ19/portfolio.git
cd portfolio
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the Development Server

```bash
npm run dev
```

Visit `http://localhost:5173/` in your browser.
